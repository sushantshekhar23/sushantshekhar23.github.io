Food Delivery

  • Tech Stack: Java, Spring Boot, MySQL, Hibernate, REST APIs, Thymeleaf, AWS EC2, Maven, Swagger/Postman
  • Github URL: Project Link

> Developed a Food Delivery System with Spring Boot, MySQL, and 40+ REST endpoints, deployed on AWS EC2.

> Developed Food, Orders, Customers, Coupons, and Delivery modules with complete CRUD functionality using Spring Boot and Hibernate.

> Implemented advanced search and filtering methods to retrieve relevant data efficiently from the database.

> Built custom queries with JPQL/HQL for order tracking, coupon management, and delivery assignments.

> Conducted API testing via Postman and Swagger to ensure robust performance.

> Optimized database interactions for better scalability and performance of the food delivery platform.

> Integrated Thymeleaf for rendering dynamic HTML pages from backend data.
